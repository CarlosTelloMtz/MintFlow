# 🌿 MintFlow

MintFlow es un centro de control "moderno" para Linux Mint XFCE.

Su objetivo es ofrecer una experiencia elegante, rápida y sencilla para usuarios nuevo.
## Características

- Hub construido con Eww
- Información del sistema en tiempo real
- Integración con Kitty
- Integración con btop
- Consejos para aprender Linux
- Diseño ligero
- Código abierto
